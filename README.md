## 📖 故事背景
歡迎來到神祕的**煉金藥水鋪**！你是這裡首席的調配師，每天都要面對來自森林各處的奇幻客人。他們需要的藥水精確度極高，多一滴太濃、少一滴太淡。

你的任務是根據客人提供的「分數」或「小數」指令，精準轉換成實驗室量筒上的刻度，守護藥水鋪的百年名聲！

## 🎯 學習目標 (符合國小四年級數學課綱)
本遊戲涵蓋以下核心能力：
* **一位、二位小數轉換**：理解 $0.1 = \frac{1}{10}$ 與 $0.01 = \frac{1}{100}$。
* **等值分數應用**：練習將 $\frac{1}{2}$、$\frac{1}{4}$ 等常用分數轉換為小數。
* **補數概念**：訓練湊成整數 1 的邏輯運算。
* **數感建立**：透過垂直分數介面，強化對分數結構的視覺記憶。

## ✨ 遊戲特色
1. **階段式挑戰**：從學徒到大師，循序漸進共有三個關卡。
2. **隨機題庫**：內建 30 題精選題庫，每局隨機挑選 2 題，保持新鮮感。
3. **友善輸入介面**：專為小學生設計的「分子/分母獨立輸入框」，直覺好上手。
4. **即時回饋**：附帶「調配小提示」與結算評語，鼓勵學生持續挑戰。

## 🛠️ 技術說明

* 無須後端伺服器，可透過 **GitHub Pages** 快速部署。
